Gem & Jewel Theme Demo

This page demonstrates the gem and jewel-based styling for the new treasure hunt.

Each page can have a different gemstone theme by adding a class to the body tag:

  • 💎 Ruby (default)
  • 💙 Sapphire: class="sapphire-theme"
  • 💚 Emerald: class="emerald-theme"
  • 💜 Amethyst: class="amethyst-theme"
  • 💛 Topaz: class="topaz-theme"
  • 💍 Diamond: class="diamond-theme"
